# RubbishReview 用户指南

> **"We only accept rubbish."** — 一个让科研人放下焦虑、拥抱失败的学术社区平台

---

## 📖 什么是 RubbishReview？

**RubbishReview** 是一个以"学术垃圾"为主题的开放投稿与同行评议社区。我们对标 OpenReview 的核心学术工作流，叠加轻社交互动，为科研人提供发表失败实验、吐槽科研日常、进行非严肃学术讨论的空间。

在传统学术出版世界里，只有"成功"的结果才能发表。但那 90% 失败的实验呢？被拒了七次的论文呢？预测结果全是噪声的神经网络呢？

**RubbishReview 为所有这些"学术垃圾"提供了一个家。**

## 🎯 核心功能

### 1. 投稿系统

支持多种内容格式：
- **Markdown** — 轻量级写作，支持 KaTeX 数学公式
- **PDF 上传** — 直接上传已编译的论文
- **Word 文档** — 支持 .doc/.docx 格式
- **图片** — 快速投稿模式，适合实验截图、失败图表

投稿流程：
1. 选择目标子刊 (Venue)
2. 填写标题、摘要、关键词
3. 上传内容（可添加补充材料）
4. 选择评审模式
5. 预览并提交

### 2. 评审系统

三种评审模式：
- **🎲 闭眼盲审** — 系统随机分配评审员
- **🌍 开放评审** — 所有人可提交评审意见
- **🚀 极速收录** — 跳过评审，直接标记为 Certified Rubbish（需编辑邀请码）

结构化评审维度：
- **垃圾程度 (Rubbish Score)** — 1-10 分，10 为顶级垃圾
- **无用程度 (Uselessness Score)** — 1-10 分，10 为对人类毫无贡献
- **娱乐价值 (Entertainment Score)** — 1-10 分，10 为笑到头掉

### 3. 评审决定

| 决定 | 含义 |
|------|------|
| 🗑️ **Certified Rubbish** | 恭喜！经认证的高质量垃圾 |
| ♻️ **Recyclable** | 有一定回收价值，建议投正刊 |
| ❌ **Too Good, Rejected** | 太好了，不符合本刊收录标准 |

### 4. 社区互动

- **评论讨论** — 支持嵌套回复的评论树
- **投票系统** — 为论文、评审、评论点赞
- **作者回复 (Rebuttal)** — 作者可对评审意见进行回复
- **分享功能** — 一键分享到社交媒体

### 5. 用户中心

- **Dashboard** — 管理你的投稿、评审和待办任务
- **个人主页** — 展示投稿历史和成就徽章
- **通知中心** — 实时接收评审结果、评论提醒
- **设置** — 个人资料和偏好设置

## 📚 子刊 (Venues)

RubbishReview 设有多个专业子刊，每个子刊聚焦特定学科领域。以下是部分代表性子刊：

| 子刊 | 戏仿对象 | 领域 |
|------|----------|------|
| **Notrue** | Nature | 主刊，综合类，收录一切学术垃圾，只发表不自然的发现 |
| **Dead Cell** | Cell | 生物，养死细胞的心路历程，只收录死掉的 |
| **Abandoned Materials** | Advanced Materials | 化学，收率低于 0.5% 的实验，只收录被抛弃的 |
| **The Fool** | NeurIPS/ICML | 计算机，跑不通的代码和过拟合的模型 |

> 💡 **更多子刊**：访问 [Venues 页面](/venues) 查看全部子刊列表。

每个子刊都有独立的：
- 投稿入口和截止日期
- 编辑团队
- 投稿统计和影响因子（永远为 0）

**Venues 是社区驱动的** —— 任何用户都可以申请创建新的子刊！访问 [申请创建 Venue](/venues/apply) 开始你的主编之旅。

## 🚀 快速开始

### 注册账号

1. 点击右上角 **Sign Up**
2. 填写邮箱和密码
3. 完善个人资料（用户名、机构、研究领域等）

### 提交你的第一篇"垃圾"

1. 点击导航栏 **Submit**
2. 选择一个正在征稿的子刊
3. 按步骤填写信息并上传内容
4. 选择评审模式（推荐新手选择"开放评审"）
5. 预览并提交

### 参与评审

1. 进入 **Dashboard** → **Tasks** 查看待审论文
2. 点击论文标题进入详情页
3. 阅读论文内容
4. 填写结构化评审表单
5. 提交评审意见

### 互动与讨论

- 在论文详情页底部参与讨论
- 为精彩的评审点赞
- 使用表情反应（🗑️💩🔥😂🏆♻️）

## 📝 投稿须知

### 什么内容适合投稿？

✅ **欢迎投稿：**
- 失败的实验和负面结果
- 被顶会/顶刊拒稿的论文
- 跑不通的代码和过拟合的模型
- 科研吐槽和自嘲
- 有趣但无用的研究想法
- 证明到一半发现漏洞的定理

❌ **不适合投稿：**
- 真正有价值的研究成果（请投正刊）
- 抄袭或剽窃内容
- 违反学术道德的内容
- 人身攻击或不当言论

### 投稿格式要求

- **标题**：不超过 200 字符
- **摘要**：不超过 500 字符
- **关键词**：最多 5 个
- **PDF/Word**：最大 50MB
- **图片**：最大 20MB/张
- **补充材料**：最大 50MB/文件

## 🆘 常见问题

### Q: 我的论文真的很好，可以投吗？

A: 不行。如果你的论文太好，会被判定为 **"Too Good, Rejected"**。请投正刊。

### Q: 评审需要多长时间？

A: 开放评审模式下，通常 1-7 天内会收到评审意见。闭眼盲审模式下，如果 14 天内无人评审，会自动转为开放评审。

### Q: 可以修改已提交的论文吗？

A: 可以。在论文详情页点击 **Edit** 进行修改，或点击 **Submit Revision** 提交新版本。

### Q: 如何成为子刊编辑？

A: 在 Venues 页面点击 **"Apply to Create a Venue"** 申请创建新子刊，或联系现有子刊编辑申请加入。

### Q: 如何获取极速收录的邀请码？

A: 联系目标子刊的编辑团队获取编辑邀请码。
